Installation
================================================
The EnsembleMoodleModule-v1.0.0.zip file has a
single directory called "ensembleContent".

1. Extract EnsembleMoodleModule-v1.0.0.zip to moodle/mod directory
2. Login to Moodle as an administrator
3. Expand the "Site administration" node and click on "Notifications"
4. You should see "Ensemble Content" module listed, waiting to be installed
5. Step through the wizard to install the "Ensemble Content" module

================================================
Configuration
================================================
1. Expand the "Site administration/Plugins/Activity modules" node
2. Click on "Manage activities"
3. Click on "Settings" in the right hand column for the "Ensemble Content" module
4. Enter the URL of your Ensemble Video web application in "Ensemble Server URL"
   (Example: https://cloud.ensembleVideo.com/ensemble)
5. Click on "Save Changes"

================================================
Usage
================================================
1. Edit a course or create a new course
2. Go to your course and click on "Turn editing on"
3. Select an area to add some Ensemble Video content and click on "Add an activity or resource"
4. Select "Ensemble Content" from under "Resources" and click "Add"
5. Enter a "Name" and "Embed Code" from Ensemble Video
6. Click on "Save and return to course"
7. Watch your Ensemble Video content in Moodle
